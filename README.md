## 실행 방법

1. `npm install`
2. `npm install -g json-server`
3. `json-server --watch wiki_db.json --port 5000`
4. `npm run dev`

- RESTful API를 구현하기 위해 json-server 라이브러리를 사용했습니다.
- json-server의 port 번호는 5000번으로 설정해두었습니다.
